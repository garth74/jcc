# JCC

This package provides a few color conversion functions that can be jit compiled using numba.

## Supported conversions

### RGB

- RGB --> HSV
- RGB --> HLS
- RGB --> XYZ
- RGB --> CIELAB

### HSV

- HSV --> RGB
- HSV --> HLS
- HSV --> XYZ
- HSV --> CIELAB

### HLS

- HLS --> RGB
- HLS --> HSV
- HLS --> XYZ
- HLS --> CIELAB

### XYZ

- XYZ --> RGB
- XYZ --> HSV
- XYZ --> HLS
- XYZ --> CIELAB

### CIE LAB

- CIELAB --> RGB
- CIELAB --> HSV
- CIELAB --> HLS
- CIELAB --> XYZ
